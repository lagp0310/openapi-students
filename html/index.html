<h3>Get All Students.</h3>

Query parameters

filter (optional)
Query Parameter — <h4>Optional parameter to filter Students.</h4>
skip (optional)
Query Parameter — <h4>Number of Students to skip.</h4> format: int32
limit (optional)
Query Parameter — <h4>Maximum number of Students to be returned.</h4> format: int32
sortBy (optional)
Query Parameter — <h4>Sort Students by one of their attributes.</h4> <h4><strong>Note</strong>: If <code>sort</code> is not present in query, <strong>ascending order</strong> is assumed.</h4>
sort (optional)
Query Parameter — <h4>Sort Students Retrieved List.</h4> <h4><strong>Note</strong>: It <strong>must</strong> be used with <code>sortBy</code>.</h4>

Return type

array[Student]
