This is the API

getAllStudents

### Get All Students.

URL

https://virtserver.swaggerhub.com/lagp0310/students-api/1.0.0/students

HTTP Method

GET

Response Type

List

Parameters

  • Query: filter String

    #### Optional parameter to filter Students.

  • Query: skip Integer

    #### Number of Students to skip.

  • Query: limit Integer

    #### Maximum number of Students to be returned.

  • Query: sortBy String

    #### Sort Students by one of their attributes. #### **Note**: If ```sort``` is not present in query, **ascending order** is assumed.

  • Query: sort String

    #### Sort Students Retrieved List. #### **Note**: It **must** be used with ```sortBy```.
