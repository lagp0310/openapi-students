# Students OpenAPI Docs.

## Getting Started.

### Prerequisites.

- Node (v12.14.1).
- NPM (v6.13.4).

### Running the Server.

First, install dependencies:
```
npm install
```

Then, just run the Server:
```
npm run docs
```

You will see a message in the Console to inform that the Server is running at: ```http://localhost:8002```.
